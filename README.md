# 🛍️ NovaZ – Full Stack E-Commerce Website

## 📌 Project Overview

NovaZ is a modern Full Stack E-Commerce web application developed using React, Express.js, MongoDB, and Node.js. It allows users to browse products, register, log in, add items to the cart, place orders, and view their order history.

The project demonstrates full-stack web development with frontend, backend, authentication, database integration, and REST APIs.

---

## ✨ Features

* 👤 User Registration & Login
* 🔐 JWT Authentication
* 🛒 Add Products to Cart
* 💳 Secure Payment Interface
* 📦 Place Orders
* 📜 Order History
* 🔍 Product Search
* 📱 Responsive User Interface
* 🗄️ MongoDB Database Integration

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/jyothilokina-blip/NovaZ-Ecommerce.git
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
