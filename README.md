# рҹҡҖ Installation

## Clone Repository

```bash
git clone https://github.com/jyothilokina-blip/NovaZ-Ecommerce.git
```

<br>

## Backend Setup

```bash
cd Backend
npm install
npm run dev
```

Server runs on:

```text
http://localhost:5000
```

<br>

## Frontend Setup

```bash
cd Frontend
npm install
npm run dev
```

Application runs on:

```text
http://localhost:5173
```
